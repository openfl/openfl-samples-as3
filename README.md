OpenFL Samples (ActionScript 3)
===========================

## Get Started

### Clone the repository

```bash
git clone https://github.com/openfl/openfl-samples-as3
```

### Choose a sample

```bash
cd openfl-samples-as3/features/display/DisplayingABitmap
```

### Install and start

```bash
npm install
npm start
```
